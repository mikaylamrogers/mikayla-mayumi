Self Portrait Stamp
( Creative Code )
p5.js
An interactive self-portrait, in which the user can dictate the final visual result of my face.
This project is an interactive self-portrait, in which the user can dictate the final visual result of my face. Each "stamp" leaves a painterly stroke as a result of its infinite movements.

This interactive program continuously responds to your clicks by adding more eyes, noses, and mouths at the end of its array. With each interaction, the self-portrait takes on a new dimension, guided by your creative choices.

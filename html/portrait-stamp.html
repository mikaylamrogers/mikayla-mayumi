Self Portrait Stamp

↪ Project Site

Tools: HTML, CSS, p5.js

This project is an interactive self-portrait, in which the user can dictate the final visual result of my face. Each "stamp" leaves a painterly stroke as a result of its infinite movements. This program was built in p5.js and continuously iterates through adding more eyes/noses/mouths at the end of its array in response to each click by the user.

Halo-Halo

↪ Project Site

Tools: HTML, CSS, Three.js

This 3D-rendered website is based on an emblem of Filipino culture, the halo-halo dessert. It is designed in the perspective of taking a deep-dive into halo-halo. Each navigational page corresponds to its three main layers of ingredients. Each layer is built via 3D models using Three.js with HTML and CSS.
